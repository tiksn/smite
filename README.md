# smite

[![Version](https://img.shields.io/nuget/v/smite-cli.svg)](https://www.nuget.org/packages/smite-cli)

DTO transpiler

## Supported Programming Languages
- [x] C#
- [x] F#
- [x] Visual Basic.NET
- [ ] TypeScript
- [ ] Python
- [ ] Java

## Install

```PowerShell
dotnet tool install --global smite-cli
```

## Help

```PowerShell
smite --help
```
